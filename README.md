## 📝 Descrição
Projeto acadêmico de banco de dados para uma empresa têxtil, contendo modelagem conceitual e lógica, mini mundo e scripts SQL completos para criação e gerenciamento do sistema, desenvolvido como atividade prática da faculdade.

## 📊 Conteúdo do Projeto
- **Mini Mundo:** Descrição detalhada do cenário de negócio.
- **Modelagem:** Diagramas Conceitual e Lógico (DER).
- **Scripts SQL:** Comandos para criação de tabelas (DDL) e manipulação de dados (DML).

## 🛠️ Tecnologias Utilizadas
- **SQL / MySQL**
- Ferramentas de Modelagem (ex: MySQL Workbench / brModelo).

## 🚀 Como Utilizar
1. Consulte a documentação do "Mini Mundo" para entender as regras de negócio.
2. Execute o script `.sql` no seu servidor de banco de dados para gerar a estrutura completa.
